Purpose

Configure an SEL-551 relay to simulate a circuit breaker with realistic opening and closing time delays. The dummy breaker is used during protection relay testing to make the relay-under-test behave as if connected to a real circuit breaker, without requiring a physical mechanical breaker in the test loop.

Design Overview

Core Concept

The SEL-551 TRIP latch is repurposed as the breaker state register:

On power-up the TRIP latch initialises to 0 — the breaker starts CLOSED, which is the correct default state for a healthy breaker.

SELOGIC Logic Design

Shared Trip/Close Timer — SV5

A single timer handles both opening and closing delays. One setting (SV5PU) controls both, and the display label always remains accurate without manual updating.

; Shared timer
SV5   = IN1 + IN2 * TRIP   ; trip cmd OR close cmd (when breaker open)
SV5PU = 5 cycles            ; 100 ms at 50 Hz
SV5DO = 0

; Trip latch
TR    = SV5T * IN1          ; latch only when IN1 caused the timer
ULTR  = SV5T * !IN1         ; unlatch only when IN2 caused the timer
TDURD = 3 cycles            ; minimum 60 ms hold time
52A   = !TRIP

Anti-Pumping

Anti-pumping is enforced at two levels. First, the IN2 * TRIP term in SV5 means the close path is naturally inactive when the breaker is already closed (TRIP=0). Second, ULTR = SV5T * !IN1 cannot fire while IN1 is held, so the close command is blocked even if the timer fires.

State Indicator Variables (SV1–SV3)

SV1 = IN1 * !TRIP           ; opening in progress → drives OUT3
SV2 = IN2 * TRIP * !IN1     ; closing in progress → drives OUT4
SV3 = IN1 * IN2 * TRIP      ; anti-pump active    → drives DP2

SV4, SV6, and SV7 are unused (= 0).

Output Equations

OUT1 = !TRIP   ; 52A — closed when breaker CLOSED
OUT2 =  TRIP   ; 52B — closed when breaker OPEN
OUT3 = SV1     ; opening indicator
OUT4 = SV2     ; closing indicator

Opening and closing use the same timer (SV5PU) and cannot be set independently. Both are always equal.

Suggested Enhancements

1. Spring Recharge Timer (Realistic Close Inhibit)

Real breakers require 5–15 seconds between successive close operations. SV6 is now unused and available:

SV6   = SV5T * !IN1         ; triggers when a close fires
SV6PU = 0.000
SV6DO = 500 cycles          ; 10 seconds at 50 Hz

SV5   = IN1 + IN2 * TRIP * !SV6T  ; block close while recharging

2. Manual TRIP/CLOSE from Front Panel

Use SEL-551 local control switches (LB1–LB8) to trip and close without external wiring:

SV5  = IN1 + LB1 + (IN2 + LB2) * TRIP
TR   = SV5T * (IN1 + LB1)
ULTR = SV5T * !(IN1 + LB1)

Configure in SET_T.TXT: NLB1 = "MAN TRIP", NLB2 = "MAN CLOSE".

3. Multi-Shot Autorecloser Simulation

The SEL-551 has a built-in 4-shot autorecloser (79 element). Configuring it would allow automatic reclosing after trips, enabling testing of relay-under-test autorecloser supervision without modifying the dummy breaker SELOGIC.

4. Breaker Failure / Lockout After N Trips

After a configurable number of consecutive trips without a successful close, lock out the breaker and require a manual reset. Partially achievable using the 79 lockout (79LO) bit from the reclosing relay.

5. SEL-551C for Additional Inputs

The SEL-551C has 6 inputs vs 2. A third input could be used for a sync-check or close-permissive signal, and further inputs for external lamp test or lockout override.
